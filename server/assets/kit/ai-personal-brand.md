## IG 帳號建立工具

Bio 寫法、品牌定位、Highlights 策略——把你的 IG 帳號變成 24 小時自動介紹你的業務員。

::: data
官方數據｜Instagram Bio 上限 150 字元。帳號 handle、名稱欄、Bio 含關鍵字，搜尋出現機率顯著提升。每週更新 Bio 連結的帳號，點擊率比靜態連結高 50%。（來源：Instagram Creator Academy / Sprout Social 2026）
:::

### Bio 結構公式（官方建議 4 行架構） {#bio-formula}

| 行數 | 用途 | 範例 |
| --- | --- | --- |
| 第 1 行 | 你是誰 + 你幫誰解決什麼問題 | 幫小型品牌用 AI 做社群行銷 |
| 第 2 行 | 社群證明或具體成果 | 協助 30+ 品牌從 0 建立 IG |
| 第 3 行 | 個人特色或差異化 | 每週分享 AI 行銷乾貨 |
| 第 4 行（CTA） | 唯一行動指令 + 連結 | 👇 免費領取品牌建立攻略 |

::: tip
關鍵字放在「名稱欄」的效果最強，因為名稱欄是 Instagram 搜尋的主要索引欄位。
:::

### 提示詞 #1 — 生成 IG Bio（5 個版本） {#bio-prompt}

::: prompt
你是一位 Instagram 個人品牌策略師。請根據以下資訊，幫我寫出 5 個不同風格的 IG Bio，每個都不超過 150 字元，並且包含明確的 CTA。
我的資訊：
- 我的職業 / 定位：[填入，例如：自由接案設計師]
- 我的目標受眾：[填入，例如：想要品牌升級的中小企業主]
- 我的核心能力：[填入，例如：品牌視覺設計、社群內容規劃]
- 我的具體成果或社群證明：[填入，例如：服務超過 50 個品牌]
- 我希望讀者做的一件事：[填入，例如：點連結看我的作品集]
請分別以以下風格各寫一個版本：
1. 專業權威型（適合 B2B）
2. 輕鬆親切型（適合生活類）
3. 數字佐證型（強調成果）
4. 問句開頭型（引發共鳴）
5. 故事型（有個人溫度） 每個版本請標示字元數。
:::

### Highlights 分類建議 {#highlights}

- **關於我** — 用 3–5 則限時動態說你的故事、背景和定位
- **作品 / 服務** — 展示代表作、客戶案例或服務項目
- **乾貨精選** — 你最受歡迎的教學或知識型貼文
- **客戶回饋** — 截圖見證、感謝訊息（建立信任）
- **免費資源** — 導流到你的電子報 / 免費下載（最重要的成長引擎）

## 電子報模板（5 種格式）

每種格式都是一個 ChatGPT 提示詞，填入你的內容就能生成整封信（含主旨列建議）。

::: data
Mailchimp / Brevo 2026｜各產業平均開信率 21–29%。使用 AI 優化主旨列，開信率提升 26%。最佳發送時間：週二、週三上午 8–10 點。每封只放 1 個 CTA，點擊率比放多個高 42%。
:::

### 模板 1 — 週報格式 {#tpl-weekly}

::: prompt
請幫我寫一封「週報格式」的電子報（分享這週學到什麼）。
品牌名：[填入]
本期主題：[填入]
這週最重要的一件事：[填入，你這週最大的學習／發現／心得]
請用繁體中文、口語、有故事感、真實不浮誇，依以下結構輸出：
1. 開頭問候讀者 ＋ 一句話點出本期主題
2.「這週最重要的一件事」：用 150–200 字分享，要有故事感
3.「3 個你可能錯過的東西」：列 3 個資源／工具／文章，各附一句推薦理由＋連結佔位
4.「本週新文章」：標題＋30 字內摘要＋閱讀連結佔位
5. 結尾署名（你的名字｜IG 帳號）
最後另外給我 2 個主旨列（≤50 字、標點不超過 2 個、不要用「免費」「限時」）。
:::

### 模板 2 — 乾貨清單格式 {#tpl-list}

::: prompt
請幫我寫一封「乾貨清單格式」的電子報（5 個工具／技巧／資源）。
本期主題：[填入，例如：5 個讓你少花 2 小時的 AI 工具]
目標讀者：[填入]
請依以下結構輸出：
1. 簡短開頭，一句話直接進入主題
2. 條列 5 個項目，每個：名稱＋一句話用途＋「為什麼值得」的具體原因＋連結佔位
3.「本期限定」：一個額外小彩蛋＋連結佔位
4. 結尾署名
最後另外給我 2 個實用型主旨列（≤50 字、不要用過度行銷詞）。
:::

### 模板 3 — 個人故事格式 {#tpl-story}

::: prompt
請幫我寫一封「個人故事格式」的電子報，讓訂閱者更認識我。
我想分享的事：[填入，一件真實發生的小事或轉折]
請用真實、不過度修飾、有畫面感的語氣，依以下結構輸出：
1. 開場：用一個具體場景或細節切入，讓讀者立刻進入畫面
2. 中段：發生了什麼、我的掙扎與轉折、我當時怎麼想
3. 結尾：一個小而具體的啟發就好
4. 邀請讀者回信互動 ＋ 署名 ＋ 一句輕鬆的 P.S.
最後另外給我 2 個帶情感、引發好奇的主旨列（≤50 字）。
:::

### 模板 4 — 促銷通知格式 {#tpl-promo}

::: prompt
請幫我寫一封「促銷通知格式」的電子報（有商品要賣的時候用）。
商品名稱：[填入]
目標受眾：[填入]
這個商品能幫他們做到：[填入 2–3 個具體成果]
優惠／限制：[填入，例如：到 X 日 ／ 限 X 個名額]
請用直接、不浮誇的語氣，且整封只放 1 個 CTA，依以下結構輸出：
1. 開頭直接說有件事想告訴你
2. 商品開放 ＋ 為誰設計
3. 條列能達成的具體成果（用 ✅）
4. 一個明確 CTA（按鈕文字＋連結佔位）
5. 給沒興趣的人一個台階 ＋ 署名 ＋ P.S. 點出名額／時間的急迫性
最後另外給我 2 個主旨列（帶適度急迫感、≤50 字）。
:::

### 模板 5 — 特別感謝格式 {#tpl-thanks}

::: prompt
請幫我寫一封「特別感謝格式」的電子報——不賣東西、不給乾貨，只是真心說謝謝。
我想感謝這群讀者的原因：[填入，真實理由]
請用真誠、不表演感動的語氣，依以下結構輸出：
1. 開頭說明這是一封只為了說謝謝的信
2. 真心話段落：為什麼感謝這群讀者
3. 讓他們知道「有你在不是理所當然」
4. 邀請他們回信告訴我「當初為什麼訂閱」 ＋ 署名
最後另外給我 2 個溫暖、個人化的主旨列（≤50 字）。
:::

### 主旨列生成提示詞 {#subject}

::: prompt
請幫我為這封電子報生成 10 個主旨列。
本期電子報主題：[填入]
目標讀者：[填入]
我希望他們有什麼感受：[好奇 / 感動 / 緊迫感 / 實用感]
規則：
- 每個不超過 50 字元
- 不要用標點符號超過 2 個
- 不要用「免費」「限時」等過度行銷詞
- 5 個用「情感型」、5 個用「實用型」
- 每個標注開信策略（好奇缺口 / 個人化 / 數字 / 問句）
:::

## 30 天執行行事曆

從 0 到 1000 粉的路徑圖。每天只做一件事，不用想，照著做就對了。遇到「發文」的日子，直接到 Part 2 內容點子庫 挑一則最適合的主題即可。

::: data
數據支持｜有清楚內容支柱 + 穩定發文節奏的帳號，前 30 天平均可累積 250–500 粉（有機成長）。週一到週四發文，比週五到週日帳號平均觸及率高 18%。（Tijus Academy 案例研究 / Buffer 2026）
:::
