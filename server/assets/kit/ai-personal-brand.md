## 帳號前置準備 {#ig-pre}

開帳號之前先把這三件事想清楚，後面所有提示詞的效果才會準。

### 提示詞 #0-A — 生成目標受眾（TA）輪廓 {#ta-profile}

::: prompt
你是一位 Instagram 個人品牌顧問。
請根據我的帳號定位，幫我生成一個具體的目標受眾（TA）輪廓。

我的帳號主題：[填入，例如：教 AI 工具技巧給上班族]
我的主要服務或產品：[填入]
我的語氣與風格：[填入，例如：輕鬆、直接、不廢話]

請輸出：
1. TA 的基本資料（年齡層、職業、生活情境）
2. TA 的核心痛點（3 個，要具體，不要籠統）
3. TA 的渴望（3 個）
4. TA 在 IG 上會搜尋什麼關鍵字？
5. TA 最常什麼時間滑 IG？
6. 一段 150 字的 TA 人物誌描述（像在描述一個真實存在的人，有名字、有生活細節）
:::

### 提示詞 #0-B — 競業分析 {#competitive-analysis}

::: prompt
幫我針對以下帳號定位，進行 IG 競業分析。

我的帳號主題：[填入]
我的目標受眾：[填入]
我的個人特色或背景：[填入]

請按以下步驟輸出：
1. 列出 5 種在台灣 IG 上與我定位相近的帳號類型（描述類型即可，不需要真實帳號名稱）
2. 這些帳號的共同做法是什麼？
3. 他們的弱點在哪裡？（我可以做得更好的地方）
4. 在這群受眾中，有哪些需求是目前沒被滿足的？
5. 給我 3 個差異化定位建議，讓我從競爭中脫穎而出
:::

### 提示詞 #0-C — 增粉可行性評估 {#feasibility}

想增粉之前，先誠實問自己這些問題。你的答案決定了你該用什麼節奏。

::: prompt
我想在 IG 上經營 [填入主題] 帳號，請幫我評估可行性並給出建議。

請根據以下我的實際狀況，誠實分析，不要說場面話：

內容素材面：
- 我現有的素材是什麼？（例如：旅遊照片、工作截圖、讀書筆記）：[填入]
- 我有沒有固定可以產出的經驗或知識？：[填入]

時間面：
- 我每週能花多少時間在內容創作上（含拍攝、剪輯、發文）？：[填入小時]
- 這個節奏我能持續 6 個月以上嗎？：[是 / 不確定]

請輸出：
1. 如果我每週只有 [填入] 小時，建議的最低可行發文頻率是什麼？
2. 以這個主題為例，我最容易在哪個環節卡關？
3. 給我一個「可長可久」的最輕量版執行方案
4. 如果我的主題是旅遊，每週可以去幾個點？預算夠嗎？拍攝、剪輯、發文能持續嗎？請用這個邏輯幫我想同類問題。
:::

### 要不要開新帳號？ {#new-account}

::: tip
重點不是帳號新舊，是你能不能長期穩定經營。

走個人 IP（用真實身份）→ 有隱私顧慮就另開，沒有的話現有帳號轉型即可。
經營產品或服務帳號 → 建議另開，定位更清晰。
最常見的誤區：帳號開了就以為成功了一半——不，開帳號是最簡單的部分。

IG 帳號就像一間商店。
你要持續曝光，才會被眾人看到。
帳號沒在更新，就像店門關著——不只演算法不推，進來的人看到過期的內容也不會留下來。
網站的更新、帳號的經營，都是同一個邏輯。
:::

## IG 帳號建立工具 {#ig-bio}

Bio 寫法、品牌定位、Highlights 策略——把你的 IG 帳號變成 24 小時自動介紹你的業務員。

::: data
官方數據｜Instagram Bio 上限 150 字元。帳號 handle、名稱欄、Bio 含關鍵字，搜尋出現機率顯著提升。每週更新 Bio 連結的帳號，點擊率比靜態連結高 50%。（來源：Instagram Creator Academy / Sprout Social 2026）
:::

### Bio 結構公式（官方建議 4 行架構） {#bio-formula}

| 行數 | 用途 | 範例 |
| --- | --- | --- |
| 第 1 行 | 你是誰 + 你幫誰解決什麼問題 | 幫小型品牌用 AI 做社群行銷 |
| 第 2 行 | 社群證明或具體成果 | 協助 30+ 品牌從 0 建立 IG |
| 第 3 行 | 個人特色或差異化 | 每週分享 AI 行銷乾貨 |
| 第 4 行（CTA） | 唯一行動指令 + 連結 | 👇 免費領取品牌建立攻略 |

::: tip
關鍵字放在「名稱欄」的效果最強，因為名稱欄是 Instagram 搜尋的主要索引欄位。
:::

### IG Bio 特殊字體 {#bio-font}

IG Bio 支援 Unicode 特殊字體，把字型轉換後直接貼進 Bio，讓帳號名稱或描述更有視覺辨識度。

**線上轉換工具：** [tw.piliapp.com/instagram/fonts/](https://tw.piliapp.com/instagram/fonts/)

| 效果 | 適合用在哪裡 | 適合定位 |
| --- | --- | --- |
| 𝗕𝗼𝗹𝗱 粗體 | 帳號名稱、關鍵詞強調 | 科技感、強調型 |
| 𝘐𝘵𝘢𝘭𝘪𝘤 斜體 | 個人品牌名、slogan | 創意、藝術、生活風格 |
| 𝕆𝕦𝕥𝕝𝕚𝕟𝕖 空心 | 分隔線、裝飾 | 活潑、年輕受眾 |

::: tip
名稱欄建議用正常字體——IG 搜尋較難索引 Unicode 特殊字體，名稱欄是搜尋主要索引位置，用正常字才不會影響被找到的機會。特殊字體用在 Bio 描述內裝飾即可。
:::

## IG 演算法與內容策略 {#ig-algo}

在開始寫腳本之前，先搞懂遊戲規則——演算法怎麼運作，決定了你的影片能被多少人看到。

### IG 演算法流量池 {#algorithm-pool}

每次發布 Reels，演算法會先推給一小批受眾測試互動率，好的話就進入下一層更大的流量池，依此類推。一萬粉之前最難，因為帳號歷史互動率低，初始流量有限。

| 流量層級 | 預估播放量 | 關鍵訊號 |
| --- | --- | --- |
| 層級 1（基礎流量） | 300–500 | 帳號本身的歷史互動率 |
| 層級 2 | 3,000 左右 | 前 3 秒停留率 |
| 層級 3 | 1.2–1.5 萬 | 完播率 + 儲存數 |
| 層級 4 | 10–12 萬 | 分享數（被傳到 DM）|
| 層級 5 | 40–60 萬 | 跨帳號分享 + 有意義的留言 |
| 層級 6 | 200–300 萬 | 全平台爆紅訊號 |
| 層級 7 | 700–1,100 萬 | 跨地區推送 |
| 層級 8 | 3,000 萬+ | 病毒式傳播 |

::: tip
一萬粉之前的核心目標：做出「高分享率」的影片。
靠觀眾幫你傳到 DM，才能突破層級 2→3 的瓶頸。
比起拼留言，被分享到 DM 是演算法最重視的訊號。
:::

### 短影音思維導圖 {#reels-mindmap}

根據你目前的粉絲數，策略應該不同：

**粉絲未達一萬 → 主攻增粉影片**

| 方向 | 操作方法 | 最終目標 |
| --- | --- | --- |
| 顧客常見問題 | 以顧客角度，用 1 分鐘列出關鍵字 → 搜尋這些關鍵字找爆款影片 | 複製爆款架構，改成自己的文案 |
| 產業常見問題 | 找同產業有影響力的帳號 → 搜尋他們的爆款影片 | 拆解架構，加入自己的觀點 |
| 經營者煩惱 | 以經營者角度列出痛點 → 找出自己的價值觀差異 | 做出有差異化的影片 |

**粉絲超過一萬 → 主攻變現影片**

| 方向 | 操作方法 |
| --- | --- |
| 個人體驗分享 | 以版主身份分享產品、服務的實際使用優勢 |
| 消費者回饋 | 找素人消費者做見證影片（比廣告更可信）|
| 互惠合作影片 | 找同量級網紅、微網紅互相推薦 |

**共同流程（不管哪個階段都適用）：**

找 3–4 部同主題爆款影片 → 拆解架構（文案、腳本、畫面節奏）→ 改寫成自己的文案 → 拍攝

### 為什麼有爆紅 Reels 卻沒有新追蹤人數？ {#viral-no-follow}

很多人發過觸及不錯的 Reels，但追蹤數幾乎沒有動。原因不是內容不好——而是觀眾點入帳號後沒有留下來。

| 陌生觀眾看完 Reels 後的行為 | 結果 |
| --- | --- |
| 直接滑走，看下一支影片 | 離開，無轉換 |
| 點入帳號 → 發現影片主題與帳號經營方向不符 | 離開，無轉換 |
| 點入帳號 → 發現版面有**資訊價值**且**主題一致性高** | ✅ 成為粉絲 |

::: tip
累積粉絲，版面需要【資訊價值】且【主題一致】。

Reels 是把陌生人帶到你門口的廣告；你的帳號主頁是那間店面。
陌生人進來的 5 秒內，他要能立刻判斷「這帳號是做什麼的、對我有沒有用」。
爆紅單片不能保證追蹤成長，帳號整體定位一致才能留人。

實際檢查清單：你的 Bio 能在 3 秒內說清楚你是誰、幫誰解決什麼問題嗎？
你的九宮格版面有統一的色調與主題方向嗎？
你的置頂貼文有放最具代表性的內容嗎？
:::

### 三大內容格式策略 {#content-formats}

IG 三種格式各有任務，不能混用：

| 格式 | 定位 | 核心邏輯 |
| --- | --- | --- |
| 🎬 **Reels（短影音）** | 陌生開發 | 主攻觸及演算法推薦的陌生受眾，負責曝光與吸粉 |
| 📋 **輪播貼文（多圖文）** | 深度轉換 | 推播給既有粉絲與被 Reels 吸引來的潛在受眾，提升收藏與專業感 |
| 💌 **限時動態（Story）** | 信任與黏著 | 專為追蹤者設計，靠互動拉近距離，建立鐵粉關係 |

**Reels（短影音）** 的關鍵：
- 影片前 3 秒必須點出痛點或製造懸疑
- 節奏明快，適合靜音觀看（必須上字幕）
- 目標：讓陌生人停下來，進而追蹤

**輪播貼文** 的關鍵：
- 第一頁（封面）：3 秒吸睛，直擊痛點
- 內頁：每張只講一個重點，字數不超過 50 字
- 最後一頁：精華懶人包 + 儲存 CTA（儲存率是帳號權重最重要的指標）

**限時動態** 的關鍵：
- 透過投票、問答框、私訊互動，讓粉絲感受到你是真實存在的人
- 幕後花絮、真實心情、翻車現場——展現「人性」比展現「完美」更有效

### 帳號信任感的核心：敢說你的觀點 {#trust-mindset}

帳號要培養的不只是追蹤者，而是鐵粉。
鐵粉的養成，來自於他們覺得「這個人說出了我心裡的話」。
你敢對世界說出你的想法，一定有同頻的人在某處等著——同頻的人，自然會靠近你。

::: tip
不要為了讓所有人喜歡你而說安全無害的話。
有觀點的帳號才有記憶點，沒有立場的帳號容易被人遺忘。
說出你對這個領域的真實想法，哪怕只有 10% 的人共鳴，這 10% 才是最忠誠的鐵粉。
:::

### 輪播圖高轉換率設計（5 個商業邏輯） {#carousel-conversion}

把從 Reels 或探索頁點進來的陌生訪客，轉化為長期追蹤者，輪播圖必須在幾秒內展現出高專業度。

**第 1 頁（封面）：3 秒吸睛，直擊痛點**

封面的唯一目標是讓人停下來。標題必須有「高相關性」或「急迫性」：
- 用數字：「經營 IG 常犯的 5 個致命錯誤」
- 用負面表述：「3 步驟，讓你的貼文收藏量翻倍」
- 標題字體要大、對比強，在九宮格縮圖中仍清晰可讀

**第 2–5 頁（內頁）：碎片化痛點，製造「這就在說我」的投射**

- 每張圖只講一個重點，字數不超過 50 字
- 多用條列式，不寫長段落
- 用情境、對話或統計數據，具體描繪痛點帶來的困擾

**第 6–7 頁（精華）：提供可立即落實的行動方案**

- 把前面的內容總結成「懶人包表格」或「Checklist」
- 目標：讓讀者覺得「這對我有用，必須存下來」
- 當陌生人按下「收藏」，演算法就會判定這是高價值貼文，進而推薦給更多人

**第 8 頁（誘餌）：製造「現在不存，以後找不到」的焦慮**

- 文案：「資訊量太大？先把這張圖存起來，下次要用隨時看！」
- 或搭配互動：「留言【IG】，私訊送你完整自媒體健檢表」
- 儲存率是觸及加分最高的指標

**最後一頁（封底）：清晰的個人品牌錨定**

- 展示個人形象照、品牌 Logo 與一句話的帳號定位
- 視覺色調必須與主頁（Grid）保持高度一致，建立信任感

### 輪播圖為什麼有效？ {#carousel-why}

輪播圖最有用的地方是**貼文格式**與**內容資訊密度**。做內容的目的是讓陌生人知道你分享什麼，並願意分享和收藏。輪播圖的優點是製作成本低、可快速產出，最適合用來增粉和養出垂直主題印象。

頁數不是核心——**每一頁都有用戶想要的內容，才是核心**。用戶願意停留翻頁，演算法就會加分。就像翻書一樣想看下去，跟短影音的完播率邏輯相同：繼續看下去，表示他從你的內容得到了什麼。

::: tip
雖然 1:1 輪播圖也可以，但有短影音的搭配時，建議統一使用直立「4:5」規格，視覺更一致，在手機上的佔版面積也更大。
:::

### 輪播圖製作規格 {#carousel-specs}

**比例規格**

| 規格 | 比例 | 適用場合 |
| --- | --- | --- |
| 直立（推薦） | **4:5**（1080×1350px） | 短影音帳號搭配使用，視覺統一 |
| 正方形 | **1:1**（1080×1080px） | 靜態圖文為主的帳號 |

**封面構圖規則**

- 封面必須保持主題一致性，色調與帳號 Grid 統一
- 人物、食物、主體要抓**九宮格構圖**，**不要佔滿畫面**
- 人物或特寫物件建議放在中上兩格，或中上四格靠左／靠右
- 這樣下方與旁邊才有空間放標題文字和短解說

**內容規則**

- 建議 **10 張以內**（超過用戶翻頁意願下降）
- 不只能放圖片，**放短影片片段效果也很好**（展示操作流程、Before/After）
- 每一頁要有**字卡＋照片**的組合，純圖少字很難傳達資訊
- 色調與封面統一，建立品牌識別感
- Hashtag 最多可放 **30 個**，建議混合大標籤（百萬+）與中小標籤（1–10 萬），放在貼文說明最末段或留言區

**輪播圖 CTA 宣傳方案**

| CTA 類型 | 範例 |
| --- | --- |
| 留言領取型 | 「留言【我也要】，私訊給你完整懶人包」 |
| 個人素材型 | 「覺得有用的話，先存起來！這種資訊以後找不到。」 |
| 聯繫資訊型 | 「想了解更多，點 Bio 連結 / 私訊我」 |

### 輪播圖標題技巧 {#carousel-title}

標題是封面最重要的元素，決定用戶要不要點開。

**版面規則**

- 每行約 **7–8 個字**，字體大小一致，不要忽大忽小
- 一個標題建議 **21 個字上下**（約 3 行）
- 字型選粗體，與背景對比要強，在九宮格縮圖中也能清楚辨認

**必包含的三個條件（至少一個）**

| 條件 | 說明 | 範例 |
| --- | --- | --- |
| 數據型 | 具體數字讓人覺得可信 | 「台灣人最不願意面對的 10 件事」 |
| 突顯特色 | 強調你的差異或門檻低 | 「不是工程師也能一天學會網站架設」 |
| 第三方認證 | 族群、名人、機構增加可信度 | 「連 ChatGPT 團隊都在用的提示詞技巧」 |
| 排比句 | 句型整齊，讀來有力 | 「省時、省力、省腦力——3 個 AI 工作流」 |
| 整合句 | 把讀者的問題直接放進標題 | 「還在手動？這 5 步讓 AI 幫你做完」 |

### Reels 黃金 3 秒 — 4 種必勝破題公式 {#golden3s}

前 3 秒只有一個目標：阻止用戶繼續往上滑。演算法非常看重「前 3 秒停留率」，這決定了影片會不會被推向更大的陌生流量池。

> 延伸參考：[鉤子公式庫 200 個](/kit/hook-formula-200)——200 個可直接套用的破題範例，依照 4 種公式分類。

**4 種破題公式**

**① 痛點直擊法**
點出受眾最頭痛、最想逃避的現況。

- 範例：「如果你每天發文觸及還是破百，先停下來！」
- 範例：「存不到錢，是因為你一直在交這 3 種智商稅。」

**② 顛覆認知法**
講出一句違反常理、讓人震驚的話。

- 範例：「別再認真經營 IG 貼文了，那只會害死你。」
- 範例：「不用運動也能瘦？這 3 個習慣被集體隱瞞了。」

**③ 結果先行法（Before / After）**
先展示最完美的最終成果，再回溯過程。

- 範例：「這是我花 3 天，把房間從垃圾屋變成北歐風的過程。」（前 3 秒直接放超美完成照）

**④ 數字具體法**
用精確的數字或時間限制，降低大腦吸收的門檻。

- 範例：「只要 3 步驟，下班後幫自己加薪 2 萬。」
- 範例：「新手花 60 秒，也能學會的 3 個調色技巧。」

**視覺與聽覺雙重 Hook（同時做這 3 件事）**

- **大標題字體：** 前 3 秒一定要有字體大、顏色對比強烈的標題字（建議黑底黃字或紅白配），打在畫面上方或中央
- **畫面動態：** 前 3 秒內至少切換 1–2 次鏡頭，大腦對移動的物體會本能保持專注
- **音效開頭：** 背景音樂在第一秒就要有節奏點，或加強烈音效（打字聲、叮咚聲、快門聲）

**前 3 秒絕對禁忌**

- ❌ 自我介紹：「嗨大家好，我是 xxx，歡迎來到我的頻道…」— 陌生人不在乎你是誰，只在乎這支影片對他有什麼好處
- ❌ 品牌 Logo 片頭：放 3 秒的 Logo 等同於直接請觀眾滑走

### Reels 製作規格 {#reels-specs}

**時長與節奏**

- 最長 **1 分 30 秒**，內容少就短，不要刻意塞長度
- 一支影片解決一個問題，節奏快、不拖泥帶水
- 靜音也能看懂：全程加字幕，字體夠大、對比強

**三段式影片結構**

| 段落 | 建議時長 | 目的 |
| --- | --- | --- |
| 開場 | 3–5 秒 | 提問、製造共鳴、或用反差吸引注意 |
| 重點內容 | 30–40 秒 | 給乾貨、解決方案、具體操作示範 |
| 鉤子 CTA | 約 10 秒 | 總結重點＋行動呼籲（追蹤、儲存、留言） |

**封面標題設計規則**

- 每行最多 **6 個字**，分 2–3 行排列，每行字數盡量一致
- 主視覺不要被標題文字擋住
- 以畫面**正中央**為主，左右要留空白
- 同時符合長方形（9:16）與正方形（1:1）的顯示格式

**置入商品的腳本邏輯**

在 Reels 裡推薦自己的產品，節奏要自然，先給價值再帶商品：

1. **開場（3–5 秒）**：提問或製造共鳴，讓觀眾有「這就是在說我」的感覺
2. **主要內容（30–40 秒）**：給乾貨或操作示範，中間自然帶到商品是「工具之一」
3. **鉤子 CTA（10 秒）**：總結重點，行動呼籲（留言領取、點 Bio 連結）

### 限時動態互動模組（4 種） {#story-modules}

Story 是經營鐵粉黏著度與高轉換率的核心場域。  
會點開限動的人，高機率已經是追蹤者，甚至對你抱有好感。  
核心邏輯：**降低思考門檻、善用 IG 內建貼紙、建立帶入感**。

| 模組 | 類型 | 核心目的 | 主要工具 |
| --- | --- | --- | --- |
| 模組一 | 輕量互動型 | 提升演算法觸及 | 投票、選擇題貼紙 |
| 模組二 | 深度交流型 | 建立黏著度與親密感 | 問答框貼紙 |
| 模組三 | 打造人設型 | 展現真實、引發投射效應 | Add Yours、Emoji 滑桿 |
| 模組四 | 行動轉化型 | 鋪陳變現漏斗 | 連結貼紙、ManyChat |

**模組一：輕量互動型（提升演算法權重）**

目的：讓粉絲不花大腦就能點擊，迅速加熱帳號觸及。

貼紙工具：「投票」或「選擇題」貼紙

主題企劃：
- **二選一對決：** 挑有共鳴的爭議話題，例如「連假要宅在家 🆚 出門看人海」
- **現況調查：** 刺中受眾痛點，例如「今天又加班的舉手 🙋 / 準時下班萬歲 🥳」

**模組二：深度交流型（建立黏著度）**

目的：透過一對一私訊，把粉絲變成朋友。

貼紙工具：「問答框」貼紙

主題企劃：
- **每週限定樹洞：** 固定在週五晚上開「這禮拜最讓你心累的一件事是什麼？」
- **讓粉絲決定你的內容：** 「明天要拍 Reels，你想看 A 主題還是 B 主題？」
- **回覆秘訣：** 粉絲回覆後，親自用帶有溫度的文字（甚至語音）回覆，這能創造極高的忠誠度

**模組三：打造人設型（展現不完美的真實生活）**

目的：輪播貼文展現專業，限動要展現「人性」，讓受眾產生投射效應。

貼紙工具：「輪到你了（Add Yours）」、Emoji 滑桿

主題企劃：
- **工作幕後花絮：** 拍包貨的過程、熬夜加班的電腦螢幕和黑咖啡、真實的心情碎碎念
- **翻車現場：** 偶爾分享不小心出糗或失敗的經驗，會更有人性，比一直展現完美更能拉近距離

**模組四：行動轉化型（暗中鋪陳變現漏斗）**

目的：當有新產品、新文章或團購要導流時使用。

貼紙工具：「連結」貼紙、私訊關鍵字觸發（可搭配 ManyChat 自動回覆）

主題企劃：
- **製造期待感（Teasing）：** 活動前 3 天就開始倒數，每天透露一點線索
- **加碼二選一：** 「想要【超簡短懶人包】還是【超神，進階密技】？投票，明天私訊給投票的人」— 精準篩選有強烈需求的潛在客戶

### 四、讓大家不覺得煩的 6 個原則 {#newsletter-retention}

::: data
Sailthru 2026 消費者調查（n=1,500）｜讀者退訂電子報的前三名原因：① 發太頻繁（46%）② 內容跟我沒關係（38%）③ 看起來像廣告（29%）。
第一名原因卻不是「沒興趣」，而是「太多」。
:::

**① 每封信只談一件事**
不要把四件事塞進一封信。一個主題 → 一個洞察 → 一個行動。讀者知道這封信在說什麼，才會開始期待下一封。

**② 寫信，不是發通知**
把電子報想成寫給一個真實朋友的信，而不是公告欄。用「你」和「我」，不要用「各位讀者」。Morning Brew 和 The Hustle 能成為百萬訂閱的原因：他們讓你覺得有個朋友在幫你整理資訊。

**③ 給比問更多（80/20 法則）**
80% 的信是純給值不賣東西，20% 才是推廣。Campaign Monitor 2026：堅守 80/20 的創作者，付費轉換率比每封都賣東西的高 3.4 倍。

**④ 讓讀者回信（雙向對話）**
每封信最後問一個問題：「你目前最卡的一步是什麼？直接回信告訴我。」讀者回信對平台演算法是強烈訊號，也讓你的信從「廣告」變成「對話」。

**⑤ 主旨列不要賣弄，要引發好奇或具體有用**
- ❌ 「這個月精彩內容來了！」
- ✅ 「我昨天試了這個，比 ChatGPT 快 3 倍」
- ✅ 「你可能搞錯了一件關於 IG 演算法的事」

**⑥ 讓訂閱者知道「取消訂閱很容易」**
聽起來反直覺，但主動說「不適合就取消，沒關係」，讀者反而更願意留下。不被尊重的訂閱者才會默默標記垃圾信。

### 五、電子報漏斗設計 {#newsletter-funnel}

最有效的個人品牌電子報漏斗不是「發信→賣東西」，而是一個**信任建立系統**：

```
IG / 部落格內容
       ↓
免費鉛磁（Free Lead Magnet）
  → PDF / 免費工具 / 免費模板
       ↓
歡迎序列（3 封自動信）
  → 信 1：歡迎 ＋ 兌現你的承諾（給免費資源）
  → 信 2：你的故事 ＋ 你能幫讀者解決什麼
  → 信 3：你的最佳內容精選 ＋ 輕推（軟性介紹你的服務或產品）
       ↓
週報（持續提供價值，建立習慣）
       ↓
不定期的促銷信（已有信任基礎，轉換率高）
```

**鉛磁設計原則（讓人願意填 Email 的交換物）：**
- 解決一個**非常具體**的小問題（不是「提升你的效率」，而是「30 分鐘完成一週 IG 排程的模板」）
- 立即可用（PDF、Notion 模板、Checklist 比影片課程更好轉換）
- 標題要包含數字或時間（「5 個」「3 分鐘」「一週」）

::: prompt
請幫我設計一個電子報歡迎序列（3 封自動信）。

我的品牌定位：[填入]
我的鉛磁（讀者訂閱時拿到的免費資源）：[填入]
我最終希望讀者做的行動（購買 / 預約 / 追蹤）：[填入]

請為每封信輸出：
1. 發送時間（訂閱後第幾天）
2. 主旨列
3. 信件結構（開頭 / 主體 / CTA）
4. 核心目標（讓讀者對我的印象是什麼）
語氣：真實、像朋友、不要像企業公關稿
:::

### 六、電子報模板（5 種格式） {#newsletter-templates}

每種格式都是一個 ChatGPT 提示詞，填入你的內容就能生成整封信（含主旨列建議）。

::: data
Mailchimp / Brevo 2026｜各產業平均開信率 21–29%。
使用 AI 優化主旨列，開信率提升 26%。
最佳發送時間：週二、週三上午 8–10 點。
每封只放 1 個 CTA，點擊率比放多個高 42%。
:::

### 模板 1 — 週報格式 {#tpl-weekly}

::: prompt
請幫我寫一封「週報格式」的電子報（分享這週學到什麼）。
品牌名：[填入]
本期主題：[填入]
這週最重要的一件事：[填入，你這週最大的學習／發現／心得]
請用繁體中文、口語、有故事感、真實不浮誇，依以下結構輸出：
1. 開頭問候讀者 ＋ 一句話點出本期主題
2.「這週最重要的一件事」：用 150–200 字分享，要有故事感
3.「3 個你可能錯過的東西」：列 3 個資源／工具／文章，各附一句推薦理由＋連結佔位
4.「本週新文章」：標題＋30 字內摘要＋閱讀連結佔位
5. 結尾署名（你的名字｜IG 帳號）
最後另外給我 2 個主旨列（≤50 字、標點不超過 2 個、不要用「免費」「限時」）。
:::

### 模板 2 — 乾貨清單格式 {#tpl-list}

::: prompt

3.「本期限定」：一個額外小彩蛋＋連結佔位
4. 結尾署名
最後另外給我 2 個實用型主旨列（≤50 字、不要用過度行銷詞）。
:::

### 模板 3 — 個人故事格式 {#tpl-story}

::: prompt

請用真實、不過度修飾、有畫面感的語氣，依以下結構輸出：
1. 開場：用一個具體場景或細節切入，讓讀者立刻進入畫面
2. 中段：發生了什麼、我的掙扎與轉折、我當時怎麼想
3. 結尾：一個小而具體的啟發就好
4. 邀請讀者回信互動 ＋ 署名 ＋ 一句輕鬆的 P.S.
最後另外給我 2 個帶情感、引發好奇的主旨列（≤50 字）。
:::

## 30 天執行行事曆 {#calendar}

從 0 到 1000 粉的路徑圖。每天只做一件事，不用想，照著做就對了。遇到「發文」的日子，直接到 Part 2 內容點子庫 挑一則最適合的主題即可。

::: data
數據支持｜有清楚內容支柱 + 穩定發文節奏的帳號，前 30 天平均可累積 250–500 粉（有機成長）。週一到週四發文，比週五到週日帳號平均觸及率高 18%。（Tijus Academy 案例研究 / Buffer 2026）
:::
